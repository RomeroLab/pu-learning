Model fitting with a full data set

Here we demonstrate basic usage of the package using sample labeled and unlabeled sequences (10,000 randomly sampled sequences from casp3 protein data sets)

Input: - path_l, path_u - WT (reference sequence) - py1

Here we use the paths where the sample txt files are saved. Please replace them with the actual file paths (e.g.Â "system.file("extdata", "sample_positive_mutations.txt", package = "pudms",mustWork = T)" -> "your dir/filename.txt").

Model fitting with training and test sets

Here, we will use 90% of the data to train a model and obtain an ROC curve with remaining 10% of the data.

  1. We split previously generated smpl_dat into 90/10 subsamples:
cv_smpl = cv_grouped_dat(grouped_dat = smpl_dat,
                         test_idx = 1,
                         nfolds = 10,
                         seed = 1) # seed for reproducibility

tr_smpl = cv_smpl$train_grouped_dat # training set
tt_smpl = cv_smpl$test_grouped_dat  # test set
  1. We fit a model using a training set tr_smpl:
cv.r = pudms(protein_dat = tr_smpl,
             py1 = protein_py1,
             order = 1,
             aggregate = T,
             basestate = NULL,
             nobs_thresh = 10)
#>  1. create a model matrix X from an aggregated dataset:

#> check number of unique factors in each position
#> obtain ``base`` amino-acid states
#> convert to the sparse one-hot-encoding model matrix

#> check whether a filtered X is a full rank matrix
#> 
#> 
#>  2. fit a model
#> Fitting 0th lambda
#> converged at 412th iterations
#> 
#> 
#>  3. compute p-values

roc = corrected_roc_curve(coef = coef(cv.r),
